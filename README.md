## Run Locally

Clone the project

```bash
  git clone https://github.com/chandreshpokiya/dashboard-node-js
```

Go to the project directory

```bash
  cd dashboard-node-js
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
